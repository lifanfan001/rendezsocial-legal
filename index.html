Rendez

Rendez is an activity-first dating app where users create and apply to 1-to-1 offline invitations. The host selects one person to meet in real life.

Users can create invitations for offline activities or browse and apply to invitations created by others. When multiple users apply to an invitation, the host reviews applications and selects one person to meet. The selected users then meet in person to participate in the activity together.

This approach prioritizes real-world connections over extended online messaging, facilitating direct meetings through shared activities and experiences.

User Data

Basic account information is used to create user profiles, enable matching between hosts and applicants, and support communication within the platform.
